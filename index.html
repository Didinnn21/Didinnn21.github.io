Profil Saya

Saya adalah mahasiswa semester 3 Teknik Informatika di IDE LPKIA Bandung, aktif di HMIF dan HIPMI. Berpengalaman di bidang teknologi dan kewirausahaan, dengan sertifikasi Microsoft Office Specialist (MOS) dan Java Programming Junior. Sekarang minat saya adalah menjadi ahli di bidang cyber security untuk berkontribusi pada keamanan digital dan kemajuan teknologi di Indonesia.

Sebagai individu yang disiplin, penuh semangat, dan memiliki komitmen kuat, saya aktif dalam berbagai organisasi, seperti Himpunan Mahasiswa Informatika (HMIF) dan Himpunan Pengusaha Muda Indonesia (HIPMI). Melalui organisasi ini, saya mengembangkan kemampuan teknis dan wawasan kewirausahaan untuk mendukung pengembangan diri secara holistik.

Saya juga memiliki beberapa sertifikasi, seperti Microsoft Office Specialist (MOS) dan Java Programming Junior, yang menunjukkan kompetensi saya di bidang teknologi. Dengan landasan pendidikan, pengalaman organisasi, dan semangat belajar yang tinggi, saya berkomitmen untuk terus tumbuh menjadi profesional yang unggul.
